# Security Policy

This project is a learning and showcase project with a strong focus on
security-aware backend architecture.

## Supported Versions

This repository does not provide production support.
Security updates are applied as part of ongoing development.

## Reporting Security Issues

If you discover a security issue or potential vulnerability, please do not
open a public issue.

Instead, contact the maintainer directly:

developer@pixelbyte.dev

All reports are appreciated and will be reviewed responsibly.

## Scope

This project does not provide legal or compliance guarantees regarding NIS2.
It is intended to demonstrate technical approaches to compliance tracking
and security-oriented design.
